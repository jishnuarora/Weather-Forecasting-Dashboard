#  Weather Forecasting App

A responsive weather forecasting web application built with HTML, CSS, and JavaScript. It fetches real-time weather data using API integration and displays temperature, humidity, wind speed, and weather conditions for any city.

## Features

* Real-time weather updates
* City-based search
* Temperature, humidity & wind details
* Responsive design
* API integration

## Technologies

* HTML5
* CSS3
* JavaScript
* OpenWeather API

## Setup

1. Get an API key from OpenWeatherMap.
2. Replace `YOUR_API_KEY_HERE` in `script.js`.
3. Open `index.html` in your browser.
